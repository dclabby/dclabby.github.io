Grid Forecaster

placeholder image

An application that uses Machine Learning models to generate forecast for Ireland's electricity grid. Forecast variables include system power generation, wind power generation, and system power demand. This work was developed as the main project for Higher Dimploma in Computing.

Grid Forecaster is an application that uses machine learning algorithms to generate forecasts for Ireland’s electricity power demand, and wind generated power. Four separate machine learning models were trained on system demand and wind generation data obtained from Eirgrid’s Smart Grid Dashboard. Time series forecasts were generated over a 24 hour period at 1 hour resolution. They were evaluated on a test data set of 1.5 years (representing 20% of the total data set). The results showed excellent accuracy for system demand, with coefficients of determination greater than 0.95 for all four models. In the case of wind generation the results were much poorer, with R 2 scores of 0.42 for three of the four models, and as low as 0.34 for one of the models. The poor results for wind generation were attributed to the fact that meteorological data (wind speed, wind direction) remains to be included as part of feature engineering (work in this area is ongoing). Given the excellent results observed for system demand, it is expected that significant improvement will be achieved for wind generation forecasts once meteorological data is incorporated in feature engineering.


Source code: github.com/dclabby/gridForecaster

Text Summarization

placeholder image

Applies extractive text summarization methods to generate automatic summaries of news articles from the Daily Mail data set. Developed for the Artificial Intelligence module of the Higher Diploma in Computing.

This project explores the application of the TextRank algorithm to the task of automatic text summarization. The TextRank algorithm ranks the importance of sentences in a text by representing the text as a graph, with vertices representing sentences and edges representing similarity between sentences. The texts to be summarized were news articles taken from the Daily Mail data set. Each text contained in the data set is accompanied by a human generated summary which may be used as reference summary against which automatically generated summaries may be assessed. Summarization was implemented according to five methods, three of which employed the TextRank algorithm. The aim of this work was to assess how the basic TextRank method compared to the simpler TF-IDF method; and to assess whether sentence embedding and unsupervised classification could be employed to further improve performance. The results suggest that the basic TextRank performed best (highest accuracy, lowest computational demand). The methods employing sentence embedding and unsupervised classification did not deliver further performance improvements.


Source code: github.com/dclabby/textSummarization

Captcha Reader

placeholder image

An application to develop and test various Convolutional Neural Network architectures for the purpose of reading simple four letter text based CAPTCHAs. This project was developed for the Machine Learning module of the Higher Diploma in Computing course.

The primary goal of this project is to develop an automated system capable of reading text based CAPTCHAs such that it tricks the system into thinking that the user is a human rather than an automated user. This will be achieved by training a convolutional neural network on labelled CAPTCHA images. The following additional goals of this project are:

  • To quantify the accuracy of the CAPTCHA reader in terms of its ability to predict individual characters, and four character CAPTCHAs;
  • To explore the effect on performance of network configuration (number of hidden layers and parameters, type of loss model, type of activation function);
  • To explore the effect of input data characteristics (number of features, number of training examples) on the tendency of the model to exhibit evidence of overfitting or undefitting.

Source code: github.com/dclabby/captchaReader
